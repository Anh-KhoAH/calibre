# CALIBRE — Calisthenics Elite

Site vitrine dédié à la calisthénie : calendrier des compétitions, 
encyclopédie des mouvements et programme hebdomadaire.

## Stack
- HTML / CSS / JS vanilla — aucune dépendance
- Un seul fichier `index.html`

## Déploiement
Hébergé via GitHub Pages.

## Contenu
- Compétitions 2025 avec statuts d'inscription
- 8 mouvements filtrables par niveau (débutant → élite)
- Programme hebdomadaire structuré
- Design sombre, typographie éditoriale
